Implementation of 
- **spatial decomposition**, 
- **temporal block decomposition** and
- **the combination of these two decompositions**
for a **Linear Quadratic Gaussian optimization problem** (LQG) that can be
decomposed  into subproblems linked by **linear coupling constraints**,
with an optimization span comprising **two time scales**.

All the subproblems appearing during the various decompositions are LQG
problems, adressed by establishing the **Riccati equation** adapted to the different
cases considered. The solution of the global problem is available by Riccati,
which allows to measure the quality of the solutions obtained  by the decompositions.

See the preprint entitled
*Space and Time Decompositions for LQG Problems in Decision-Hazard Information Structure*
